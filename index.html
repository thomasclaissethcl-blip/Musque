Improviser comme un laboratoire sonore

Chaque capsule propose une consigne instrumentale à tester chez soi, deux séquences exemples jouées dans le navigateur, des points clés révisables, puis des exercices auto-corrigés. Le tout reste 100 % statique et hébergeable sur GitHub Pages.

0 / 100 XP vers le niveau suivant

Lecteur intégré

Les exemples sont joués par synthèse locale dans le navigateur. Aucun backend ni fichier audio externe n’est nécessaire.

Ateliers

Étape : découverte

Le parcours guidé débloque les ateliers dans l’ordre prévu pour chaque famille de techniques.

Révision espacée

Chaque point clé mémorable devient une carte à revoir. Le système reste volontairement simple pour démontrer le potentiel du produit final.

Exercice rapide

Le moteur tire un exercice dans le périmètre actif. Les types utilisés ici sont QCU, QCM, vrai/faux, texte court, appariement et remise en ordre.
